==============================================================================================================================================================================================
											Configuring Computer Devices Before Accessing The System
==============================================================================================================================================================================================

This is a guide to set up your device to ensure the system can be accessed successfully.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Setting up a localhost server
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
1. Download XAMPP software from the official website.
2. Install XAMPP software based on the instruction given.
3. Done.


----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Install Programming Tool
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
1. Install any coding tools such as Visual Studio Code.
2. Click the "Download" button to install the packages of the software.
3. Install the software into the computer device based on the instructions given. 
4. Done.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Install Laravel Framework
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
1. Install Laravel software packages from the official website.
2. Click the "Download" button to install the packages of the software.
3. Install the software into the computer device based on the instructions given. 
4. Done.


----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Install Composer
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
1. Install Composer software packages from the official website.
2. Click the "Download" button to install the packages of the software.
3. Install the software into the computer device based on the instructions given. 
4. Done.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Access Water Mangement Plant Information System folder
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
1. Download "WMS" folder that contains source code of the proposed system.
2. Extract the folder under the "htdocs" of xampp software package that installed in the computer device before.
3. Drag the "WMS" folder into Visual Studio Code. 
4. Run the command "npm run dev" in the terminal of the folder in VS Code.
5. After the command is running successfully, then run the command "php artisan serve".
6. Then, press "CTRL+Click" the url given for the proposed water management system.
7. The link lead the user to the localhost proposed water management system. 

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Register An Account In Proposed Water Mangement Plant Information System
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
1. Click "Register" option at the top-right corner of the page to register an user account.
2. In the register page, enter the neccesary information for registration.
3. Click "Submit" to complete the registration step.
4. Done.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Login An Account In Proposed Water Mangement Plant Information System
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
1. Click "Login" option at the top-right corner of the page to login an registered user account.
2. In the login page, enter the neccesary registered email address and password to log into the account registered before.
3. Click "Submit" to complete the login step.
4. Done.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Manage User Information In Proposed Water Mangement Plant Information System
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
1. Click "Manage User" option at the navigation above to manage user account.
2. In the manage user page, the user can click "Add New User" to add new user account, "Show" to view the user account information, "Edit" to edit user information, and "Delete" to delete entire user information..
3. Done.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Manage Role Information In Proposed Water Mangement Plant Information System
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
1. Click "Manage Role" option at the navigation above to manage user account.
2. In the manage user page, the user can click "Add New Role" to add new role for the user account, "Show" to view the role information, "Edit" to edit role information, and "Delete" to delete entire role information..
3. Done.


----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Logout From Curent Account In Proposed Water Mangement Plant Information System
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
1. Click the username displayed at the navigation bar above.
2. Click the dropdown option displayed "Logout".
3. The user redirect to the home page of the proposed water management system.
4. The user is logged out.
5. Done
